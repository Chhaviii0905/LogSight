# LogSight

LogSight is a full-stack Application Monitoring and Observability Platform built using Angular 18, .NET 8, SQL Server, and MongoDB.

## Tech Stack

- Angular 18
- .NET 8 Web API
- SQL Server
- MongoDB
- Entity Framework Core
- JWT Authentication
- SignalR
- Serilog

## Features

- Request Logging
- Exception Tracking
- Performance Monitoring
- Dashboard Analytics
- Alert Rules
- Real-time Notifications
